Round 7 — Ecosystem Infrastructure

New Components (3)

  • ResourceInventory — bounded multi-resource container with deposit/withdraw
  • GraphNode — entity-to-entity graph edges with typed labels and weights
  • VisualConnection — visual links (line, dashed, catenary, flow) between entities

New Engine Modules (4)

  • FlowNetwork — directed graph of resource flow edges with priority-based transfer
  • ProceduralGen — seeded RNG, 2D value noise, cellular automata caves, dungeon generator
  • EnvironmentClock — cyclical time system (day/night, seasons) with phase queries
  • DensityField — 2D scalar field with bilinear interpolation and Jacobi diffusion

Round 6 — Feature Bonanza

New Components (2)

  • SpriteAnimator — named animation clips with per-frame duration, looping, speed
  • PhysicsJoint — distance, spring, rope, hinge joints with break force

New Engine Modules (4)

  • EventBus — channel-based typed event system with source/target entities
  • InputMap — abstract input layer mapping raw keys to named actions/axes
  • Pathfinding — A* grid pathfinding with octile heuristic, diagonal movement
  • Save/Load — world state serialization to JSON with selective restore

Round 5 — Expert Review & E2E Testing

  • Ghost trail ring buffer O(1) fix
  • Raycast DDA entry t-value fix

  • 22 comprehensive E2E integration tests

Round 4 — Hierarchy, FSM, Coroutines, TileMap

New Components (5)

  • Parent/Children/WorldTransform — entity hierarchy with transform propagation
  • StateMachine — data-driven FSM with time/signal/state transitions
  • Coroutine — scripted async step sequences (wait, set state, spawn, log)

New Engine Modules (4)

  • TileMap — row-major grid with viewport-culled rendering
  • Raycast — ray-circle, ray-AABB, DDA grid traversal
  • SpatialHashGrid — cell-bucketed spatial index for radius/AABB queries
  • EntityPool — pre-warmed entity recycling with named pools

Round 3 — Tweens, Signals, Time, Screen FX

  • PropertyTween — 9 easing curves for any numeric property, looping/ping-pong
  • EntityFlash — hit flash, blink, color pulse with auto-expiry
  • GhostTrail — fading afterimage ring buffer
  • TimeScale — per-entity time multiplier (frozen/slow-mo/fast)
  • Active — entity enable/disable flag
  • WaypointPath — once/loop/ping-pong path following
  • SignalEmitter/SignalReceiver — wired logic gates with AND/OR/edge detection
  • ScreenFxStack — composable tint/desaturate/flash effects
  • SceneManager — named scene registry with push/pop stack

Round 2 — Camera, Sprites, Physics Materials

  • Sprite Renderer — sprite sheet support with frame selection
  • Scene Transitions — fade, iris, pixelate effects
  • PhysicsMaterial — friction, drag, bounciness per entity
  • Impulse — one-shot force application
  • MotionConstraint — speed cap, axis lock
  • ZoneEffect — wind, drag, conveyor area effects
  • DialogueQueue — dialogue boxes, notifications, floating text

Round 1 — Space Survival

Foundation
