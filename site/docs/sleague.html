S-League Game Module

Minigolf RPG demo. Pokemon-style overworld with random encounter minigolf fights. 480×720 portrait, touch + mouse.

Overview

S-League is a two-mode game: a top-down overworld where you walk through grass and trigger encounters, and a fight screen where you play minigolf against monsters. Sink the ball in par or less to win XP and level up.

The entire game is a single Rust module (sleague.rs, ~1100 lines) that integrates with the engine via function pointers. No ECS entities are used — all game state lives in engine.global_state (a key-value store).

Game Modes

Overworld (mode = 0.0)

30×37 tilemap. Tap to move one tile at a time (cardinal directions). Walk through wild grass for a 25% encounter chance per step. Town center with buildings and paths. Water and trees block movement.

Fight / Minigolf (mode = 1.0)

Slingshot aiming: drag away from ball, release to launch. Ball bounces off walls with 0.65 restitution. Sand tiles triple drag. Sink the ball before max strokes to win. 4 physics substeps per frame.

Physics

ConstantValuePurpose
BALL_RADIUS5.0 pxCollision radius
HOLE_RADIUS8.0 pxSink detection radius
MAX_POWER400.0 px/sMaximum launch velocity
DRAG1.8Drag coefficient (3× on sand)
RESTITUTION0.65Wall bounce energy retention
STOP_THRESHOLD8.0 px/sSpeed below which ball stops

Physics Loop (per frame)

  1. 4 substeps per frame (sub_dt = dt/4)
  2. Read tile under ball — sand tiles multiply drag by 3
  3. Apply drag: vx -= vx * drag * sub_dt
  4. Compute new position: new_x = bx + vx * sub_dt
  5. Check 5 collision points (center + 4 cardinal offsets by BALL_RADIUS) against tilemap solids
  6. Resolve: reverse velocity component(s) × RESTITUTION
  7. Wall bounce: screen shake proportional to impact speed, increment bounce counter
  8. Hole check: if distance < HOLE_RADIUS and speed < 250, ball sinks (win)

Scoring & Progression

XP Multiplier (based on strokes vs par)

Strokes vs ParMultiplierLabel
≤ -2 (eagle)3×+1 HP
-1 (birdie)2×—

+1 (bogey)0.5×-1 HP
>+10.25×-1 to -3 HP

Level up when XP ≥ level² × 50. Even levels increase max HP by 1. Failed fights (too many strokes) cost 3 HP. Mercy heal restores full HP when HP drops to 0.

Public API

// Setup

// Game loop

// Headless action dispatch
pub fn dispatch_action(engine: &mut Engine, action: &ScheduledAction)

// AI scoring functions (operate on SimResult, not Engine)
pub fn score_hole_completion(sim: &SimResult) -> f64   // 1.0 if sunk

pub fn score_proximity_to_hole(sim: &SimResult) -> f64  // 1 - dist/500

Headless Testing Integration

S-League provides all 4 function pointers needed by the headless infrastructure:

ScenarioBuilder::new(
    sleague::setup_fight_only,   // setup
    sleague::update,             // update
    sleague::render,             // render
    sleague::dispatch_action,    // action dispatch
)

The three scoring functions (score_hole_completion, score_stroke_efficiency, score_proximity_to_hole) plug directly into FitnessEvaluator for AI-driven optimization.

The fight-specific state keys (dist_to_hole, best_dist, wall_bounces) were added specifically for AI observability, enabling anomaly detection and trajectory analysis.
